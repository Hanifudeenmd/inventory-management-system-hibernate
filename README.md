# Hibernate Inventory Management System

This project is a Java-based Inventory Management System developed using Hibernate ORM.
It is designed to manage product details efficiently with proper database persistence.

## Features
- Add new products to inventory
- Update product details (price, quantity, category, status)
- Delete products from inventory
- View all available products
- Search products by category and availability status
- Hibernate ORM for database operations (no manual SQL)

## Technologies Used
- Java
- Hibernate ORM
- JPA
- Maven
- MySQL / PostgreSQL
- Eclipse IDE

## Project Structure
- Entity classes for mapping database tables
- DAO layer for CRUD operations
- Hibernate configuration using `persistence.xml`
- Console-based application

## How to Run the Project
1. Clone or download the repository
2. Import the project into Eclipse as a Maven project
3. Configure database details in `persistence.xml`
4. Run the main class to perform inventory operations
